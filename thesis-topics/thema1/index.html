Konversationsbasierte Komfort-Navigation

Mehrere Komfortindikatoren und eine Echtzeit-Schattensimulation werden kombiniert, um fußgängerfreundliche Routen zu identifizieren. Das Projekt integriert bestehende Arbeiten und bietet ein LLM-gestütztes Chatbot-Interface: konversationelle Abfragen, Präferenzen erfassen, interaktive Routenanpassung.

English Summary

This thesis combines multiple comfort indicators and real-time shadow simulation to identify pedestrian-friendly routes. The core goal is to build an LLM-powered conversational interface that captures user preferences and supports interactive route adaptation in urban navigation scenarios.
