Thesis topics

Below is a (growing) list of Bachelor and Master thesis topics. If you are interested, please contact me at yu.feng@hs-mainz.de or y.feng@tum.de.

  • Topic 1 — Conversational comfort-based navigation — fuse multiple comfort indicators and real-time shadow simulation to identify pedestrian-friendly routes. The project will integrate prior work and provide an LLM-driven chatbot user interface (conversational queries, preference elicitation, interactive route refinement). Relevant prior work:
    • Beyond Distance: Integrating Walkability Indicators for User-Centered Pedestrian Navigation
    • Walking in the Shade: Shadow-oriented Navigation for Pedestrians
